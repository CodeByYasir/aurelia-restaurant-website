## ✨ Overview

**Aurelia Restaurant Website** is a clean and stylish static website project for a restaurant brand.  
It includes dedicated pages for menu browsing, reservations, gallery, private dining, gift cards, and more — giving visitors a complete digital restaurant experience.

---

## 🚀 Features

- ✅ Multi-page responsive restaurant website
- ✅ Dedicated pages for core restaurant sections
- ✅ Organized structure with separate CSS and JS folders
- ✅ Easy to customize branding, text, and images
- ✅ Lightweight and fast (no heavy framework)

---

- `index.html` — Home

## ▶️ Getting Started

Since this is a static website, you can run it instantly.

### Option 1: Open directly
1. Clone the repo
2. Open `index.html` in your browser

### Option 2: Use Live Server (recommended)
If you use VS Code:
1. Install the **Live Server** extension
2. Right-click `index.html`
3. Select **Open with Live Server**
